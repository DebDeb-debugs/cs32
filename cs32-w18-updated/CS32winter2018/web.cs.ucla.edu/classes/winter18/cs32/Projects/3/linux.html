Building NachenBlaster under Linux

This is how to build a soundless version of NachenBlaster, assuming you've already installed freeGLUT:

  1. Download the soundless Linux skeleton for NachenBlaster.

  2. To build the program, change into the NachenBlaster directory and type 

    	make

  3. To run the program, type 

    	./NachenBlaster
